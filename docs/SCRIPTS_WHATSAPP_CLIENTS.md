# Scripts WhatsApp - GLOBAL TIC PrintTech

Bibliotheque de messages pre-rediges pour la communication client.
Chaque script contient des variables dynamiques entre crochets a remplacer avant envoi.

**Regles :**
- Ne jamais envoyer automatiquement
- Toujours adapter au contexte
- Rester professionnel, court et rassurant
- Un seul message = une seule information

---

## Variables disponibles

| Variable | Source |
|----------|--------|
| `[Nom client]` | Champ `contact_name` du client ou `full_name` du prospect |
| `[Prenom commercial]` | Nom du commercial connecte |
| `[Reference devis]` | Format DEV-XXXX |
| `[Reference commande]` | Format CMD-XXXX |
| `[Montant]` | Total TTC formate (ex: 150 000 FCFA) |
| `[Solde]` | `total - paid_amount` |
| `[Date livraison]` | Champ `estimated_delivery` |
| `[Date validite]` | Champ `valid_until` du devis |
| `[Adresse]` | Champ `delivery_address` |
| `[Produit]` | Nom du produit commande |
| `[Delai]` | Delai estime de production |
| `[Lien Google Business]` | URL de la fiche Google GLOBAL TIC |

## Bonnes pratiques

1. **Personnaliser** : Toujours utiliser le prenom/nom du client
2. **Un message = un sujet** : Ne pas melanger devis et livraison
3. **Horaires** : Envoyer entre 8h et 19h uniquement
4. **Repondre vite** : Un prospect froid est un prospect perdu
5. **Confirmer** : Toujours accuser reception des fichiers recus
6. **Ne pas spammer** : Maximum 1 relance par jour
7. **Ton** : Professionnel mais chaleureux, pas de familiarite
8. **Emojis** : Utiliser avec moderation (1-2 max par message)
9. **Vocal** : Preferer l'ecrit pour garder une trace
10. **Urgences** : En cas de retard ou probleme, prevenir le client AVANT qu'il ne demande
